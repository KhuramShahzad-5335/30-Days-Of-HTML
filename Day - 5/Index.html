Semantic HTML Tags

Introduced many semantic tags.

Semantic tags are defined as:

Semantic HTML elements are those that clearly describe their meaning
in a human- and machine-readable way.

List of new semantic elements.

The semantic elements added in HTML - 5 are :

  • Article
  • Aside
  • Details
  • Figcaption
  • Figure
  • Footer
  • Header
  • Main
  • Mark
  • Nav
  • Section
  • Summary
  • Time
